Figure above is the overview of how Gatsby works.

Gatsby works at two phases: build time and run time.

## Build Time

Source Plugin

- API (REST/GraphQL)
- filesystems
- database

Transformer Plugin

- parsing/transformation
- optimization

GraphQL to pull those data as props of React components

Prerender and generate html, images, sitemap etc.

## Run Time

Serve the pre-render html and then hydrate it

Prefetching

Lazy Loading
